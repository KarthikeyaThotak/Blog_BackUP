Released Date: 11/27/2022

Group elements of same Indices in Python3: Let's get started

You will originally have two or more lists inside of a list like [[a, b], [c, d]] to group entries of the same index. You must make two new lists and store the items from both lists at indexes 0 [a, c] and 1 [b, d] in order to group the components of these lists. Grouping the components of the same indices has this purpose.

#Python3.10

# Project: Grouping Same Indices


# Array  of Data
InputList = [[10, 40, 50], [20, 70, 30], [90,30,60]]
OutputList = []
index = 0

for i in range(len(InputList[0])):
    OutputList.append([])
    for j in range(len(InputList)):
        OutputList[index].append(InputList[j][index])
    index = index +1

a, b, c = OutputList[0], OutputList[1], OutputList[2]
print(a,b,c)

Result:
[2, 6, 10, 13, 14, 15, 16, 17, 19]


Summary

You must first build two new lists and store the members of both lists at indexes 0 [a, c] and 1 [b, d] in order to group items with the same indices. An example of this would be [[a, b], [c, d]].
